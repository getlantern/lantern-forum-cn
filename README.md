# Lantern 中文论坛

您找到了 Lantern蓝灯 官方中文论坛！

## 有问题需要官方解决？
2025年底起，我们不再通过GitHub系统性回答问题（[为什么？](https://github.com/getlantern/lantern-forum-cn/discussions/1540)）。

如果您有问题需要官方解决，请联系：
1. 首选方案：请在 App 内提交工单（会自动附带诊断日志与设备信息）以便我们快速解决。
2. [在线表格](https://support.lantern.io/)
3. 邮箱[support@getlantern.org](mailto:support@getlantern.org)

## 那要论坛做什么？
- 我们仍然通过论坛发布动态、Beta测试链接。
- 如果不是特别忙，我们也会在论坛上参与讨论。
- 大家也可以随便说说关于Lantern、机场、VPN等等的体验。请尊重彼此，并遵循[行为守则](https://github.com/getlantern/lantern-forum-en/blob/main/conduct.md)。
